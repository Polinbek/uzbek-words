# uzbek-words
Ўзбекча сўзлар рўйхати

Сўз туркумлари сўздан кейин битта <tab> белигиси ташлаб ёзилган. Агар бир сўз
бир неча сўз туркумига тегишли бўлса, сўз туркумлари вергул орқали ажратиб
ёзилган.

## Қисқартмалар
* о - от
* с - сифат
* сн - сон
* ол - олмош
* ф - феьл
* р - равиш
* к - кўмакчи
* б - боғловчи
* ю - юклама
* у - ундов сўз
* т - тақлид сўз
* м - модал сўзлар

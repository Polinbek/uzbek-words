# uzbek-words
 List of words in the Uzbek language

These words can be used to create Uzbek wordnet, spelling or other kinds of dictionaries

####Abbreviations
* о - от
* сф - сифат
* сн - сон
* ол - олмош 
* ф - феъл
* р - равиш
* к - кўмакчи
* б - боғловчи
* ю - юклама
* у - ундов сўз 
* т - тақлид сўз
* м - модал сўз
